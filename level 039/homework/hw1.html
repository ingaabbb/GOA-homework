Javascript-ის შესახებ

Javascript-ის შესახებ: უსაფრთხოება: javascript და html:

Javascript-ის შესახებ:

Javascript შეიქმნა იმისათვის, რომ "გაეცოცხლებინათ" ანუ უფრო დინამიური გაეხადათ HTML გვერდები. ამ ენას თავდაპირველად ეწოდებოდა LiveScript, მაგრამ მაშინ, ისევე როგორც ახლა, ძალიან პოპულარული ენა იყო Java და მარკეტოლოგებმა გადაწყვიტეს, რომ Java-სთან მიმსგავსებული დასახელება უფრო პოპულარულს გახდიდა ენას. ასე წარმოიშვა სახელი javascript. ჩაფიქრებული იყო, რომ javascript ყოფილიყო java-ს შვილობილი ენა, მაგრამ მოვლენები სხვანაირად განვითარდა, javascript დაიხვეწა, განვითარდა და ჩამოყალიბდა დამოუკიდებელ ენად, რომელსაც გააჩნია თავისი სპეციფიკაცია - ECMAScript.

უსაფრთხოება:

JavaScript-ს აქვს უსაფრთხოების რისკები, რადგან მავნე კოდი შეიძლება ბრაუზერში გაეშვას. ამის თავიდან ასაცილებლად: სკრიპტები მუშაობენ sandbox-ში, სადაც მხოლოდ ვებთან დაკავშირებული მოქმედებებია დაშვებული. მოქმედებს same-origin policy, ანუ ერთი საიტის სკრიპტს არ აქვს წვდომა მეორეზე (მაგ. ქუქიებზე, პაროლებზე).დამატებით გამოიყენება უსაფრთხო ვერსიები (ADsafe, SES), ინსტრუმენტები (Closure Toolkit) და Content Security Policy, რომელიც ზღუდავს, რომ მხოლოდ სანდო კოდი გაეშვას

javascript და html:

JavaScript შეიძლება იყოს HTML-ის შემადგენელი ნაწილი და მოთავსებული იყოს ნებისმიერ ადგილას, შეუზღუდავი რაოდენობით. იგი ყოველთვის მოთავსებულია ტეგებს შორის, HTML 4.01 სტანდარტის მიხედვით, type="text/javascript" ტეგის საშუალებით.
